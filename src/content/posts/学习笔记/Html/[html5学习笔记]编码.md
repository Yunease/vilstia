# HTML5 学习笔记：编码

## 基本概念

所有数据在电脑中储存都是按照**二进制**储存，我们的数据会被转化成二进制编码。当我们读取这一段文字的时候，会把二进制编码转化成字符查看。

### 两个过程

- **编码**：将字符转换成二进制代码
- **解码**：将二进制代码转换成字符

### 字符集

编码和解码的时候需要遵守的一定规则，被称为**字符集**。

### 乱码

当编码和解码采用的字符集**不同**的时候，就会产生乱码，实际上是因为不对应产生的。

## 常见字符集

| 字符集 | 说明 |
|--------|------|
| ASCII | 美国标准字符集 |
| GBK | 中国标准字符集扩展字符集 |
| GB2312 | 中国标准字符集 |
| ISO88591 | 欧洲标准字符集（法语，德语等） |
| UTF-8 | 万国码，包含所有国家的语言符号，支持任何一种语言 |

> 开发程序的时候**最好使用 UTF-8 字符集**。
