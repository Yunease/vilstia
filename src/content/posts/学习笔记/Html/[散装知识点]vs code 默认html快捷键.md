# 散装知识点：VS Code 默认 HTML 快捷键

## 系统快捷键

| 快捷键 | 功能 |
|--------|------|
| `Windows + d` | 全部程序最小化（保留桌面） |

## 文件操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + s` | 快速保存 |

## 代码编辑

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + /` | 快速注释 |
| `Ctrl + alt + ↓` | 将一行代码向下复制 |
| `Shift + alt + ↓` | 把本行的内容向下复制一行（↑ 就是向上复制） |
| `Ctrl + shift + 回车` | 不管在哪里都向上新建行，不打乱文本 |
| `Ctrl + 回车` | 不管在哪里都向下新建行，不打乱文本 |

## 快速生成标签

### `$` 符号

标签 `$*数字` = 重复几次标签，`$` 表示从上往下数：

`h$*3` 的结果：

```html
<h1></h1>
<h2></h2>
<h3></h3>
```

### 带内容的生成

`h$*3{我是$级标题}`

```html
<h1>我是1级标签</h1>
<h2>我是2级标签</h2>
<h3>我是3级标签</h3>
```

> 在文本之内的 `$` 会被替换成数字

### 纯重复生成

如果不想要替换，可以用 `*` 来处理：

`h1*3` 的结果：

```html
<h1></h1>
<h1></h1>
<h1></h1>
```

### img 标签

输入 `img` 和回车可以快捷出现 img 标签并且附带两个属性框：

```html
<img src="" alt="">
```
