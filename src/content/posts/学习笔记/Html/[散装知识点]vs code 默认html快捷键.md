# [散装知识点]vs code 默认html快捷键

Windows + d = 全部程序最小化（保留桌面）

Ctrl + s = 快速保存

Ctrl + / = 快速注释

标签$*数字 = 重复几次标签，并且$表示从上往下数，就比如说：

h$*3 在编辑器的获得结果就是这样的：

```html
<h1></h1>
<h2></h2>
<h3></h3>

并且可以执行这样的操作：

h$*3{我是$级标题}

大括号的内容就是标签中的字符，生成的结果在三个排列的标签的基础之上，还多了内置排序，可以理解为c语言的循环结构，输出结果是这样的：

<h1>我是1级标签</h1>
<h2>我是2级标签</h2>
<h3>我是3级标签</h3>

在文本之内的$会被替换成数字。

如果不想要替换的话，就可以直接用简洁的*来处理快捷键：

H1*3

的结果就是：

<h1></h1>
<h1></h1>
<h1></h1>

Ctrl + alt + ↓ = 将一行代码向下复制

在浏览器中按下F12会打开开发者工具，可以查看源代码和对应的区域。

Ctrl + shift + 回车 = 不管在哪里都向上新建行，不打乱文本。

Ctrl + 回车 = 不管在哪里都向下新建行，不打乱文本。

输入img和回车可以快捷出现img标签并且附带两个属性框：

<img src="" alt="">

Ctrl + shift + delete(在删除键的上面) = 可以在浏览器中删除访问记录。

Shift+alt+↓= 把本行的内容向下复制一行。（↑就是向上复制）

Crtl+b = 删除光标所在行
