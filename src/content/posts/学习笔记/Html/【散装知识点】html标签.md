# 〖散装知识点〗html标签

Html标签分为两种：

双标签：

```html
<x></x>
```

单标签：

```
<x/>
```

单标签是可以省略 / 的，也就是可以写成这个样子：

```
<x>
```

标签也可以进行嵌套：

```
<x>
      <y>
      内容
      <y>
<x>
```

这种情况下，外层的就是内层的父标签，y标签就是x标签的子标签。

属性的作用是描述标签。

属性=”属性值”

属性要写在开始标签的地方，比如：

```
<开始标签 属性=”属性值”>内容</结束标签>
```

一个标签可以包含多个属性，不同属性之间只需要用空格隔开就可以了（空格不限，但是至少要一个），比如：

```
<开始标签 属性1=”属性值1” 属性2=”属性值2” 属性3=”属性值3”>内容</结束标签>
```

这些属性之间没有顺序关系随便写就行了，写成这样是完全等价的：

```
<开始标签 属性3=”属性值3” 属性2=”属性值2” 属性1=”属性值1”>内容</结束标签>
```

标签$*数字 = 重复几次标签，并且$表示从上往下数，就比如说：

h$*3 在编辑器的获得结果就是这样的：

```
<h1></h1>
<h2></h2>
<h3></h3>
```

并且可以执行这样的操作：

h$*3{我是$级标题}

大括号的内容就是标签中的字符，生成的结果在三个排列的标签的基础之上，还多了内置排序，可以理解为c语言的循环结构，输出结果是这样的：

```
<h1>我是1级标签</h1>
<h2>我是2级标签</h2>
<h3>我是3级标签</h3>
```

在文本之内的$会被替换成数字。

在浏览器内，如果鼠标无法选中，可以使用 alt + 鼠标 来选中文字复制。

图片标签 img 是一个单标签。
