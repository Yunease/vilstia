# 散装知识点：HTML 标签

## 标签分类

### 双标签

```html
<x></x>
```

### 单标签

```html
<x/>
```

> 单标签是可以省略 `/` 的，也就是可以写成：`<x>`

## 标签嵌套

标签也可以进行嵌套：

```html
<x>
    <y>
        内容
    </y>
</x>
```

这种情况下：
- 外层的就是内层的**父标签**
- y 标签就是 x 标签的**子标签**

## 属性

**属性的作用**：描述标签

### 语法格式

```html
属性="属性值"
```

### 属性写法

- 属性要写在开始标签的地方
- 一个标签可以包含多个属性，不同属性之间只需要用**空格**隔开就可以了（空格不限，至少要一个）

```html
<开始标签 属性1="属性值1" 属性2="属性值2" 属性3="属性值3">内容</结束标签>
```

> 这些属性之间**没有顺序关系**，随便写就行了，完全等价

## 快捷生成语法

### `$` 符号

标签 `$*数字` = 重复几次标签，并且 `$` 表示从上往下数：

`h$*3` 在编辑器的获得结果：

```html
<h1></h1>
<h2></h2>
<h3></h3>
```

### 带内容的快捷生成

`h$*3{我是$级标题}`

```html
<h1>我是1级标签</h1>
<h2>我是2级标签</h2>
<h3>我是3级标签</h3>
```

> 大括号的内容就是标签中的字符
> 在文本之内的 `$` 会被替换成数字

## 浏览器技巧

- 在浏览器内，如果鼠标无法选中，可以使用 **Alt + 鼠标** 来选中文字复制
