# 联合

## 基本定义

联合在写法上和结构体非常相似：

```c
union a {
    int a;
    char c;
} x1, x2;
```

使用的时候也是用`.`来使用的：

```c
x1.a = 1;
x1.c = 'w';
```

## 与结构体的区别

和结构体不同的是，每个结构体单独使用一个独立的字节空间，但是联合是共同占据，就像是这些数据联合使用了一块内存。

当你填入`x1.a=1`的时候，就已经占据了这个联合的内存空间，如果再赋值`x1.c='w'`，那么又会覆盖x1.a的内存。
