# CSS 基本语法

## 基本结构

CSS 的基本语法分为两个部分：**选择器** 和 **声明块**

```css
p {
    color: red;
}
```

- `p` 就是**选择器**
- 大括号的内容就是**声明块**

## 选择器

- 作用：选中页面的指定元素
- 示例中的 `p` 可以选中页面的所有 `<p>` 标签

## 声明块

- 作用：指定要为元素设定的样式
- 结构：**名值对应结构**
  - 一个样式名对应一个样式值
  - 名和值之间以 `:` 链接
  - 以 `;` 结尾

> 类似 C 语言的语法，结尾加上分号。
