元素选择器：

作用：根据标签名来选中指定的元素

语法：就是标签名

例子：p{}

缺点：一变全都变

Id选择器：

作用：通过元素的id属性选择一个元素。

语法：#id属性值

例子：#box{}

缺点：一次只能操作一个元素，而且很容易和JavaScript重合。

接下来有一个办法可以替代：

Class属性是一个html的属性，可以代替id使用，并且class是可以重复的，比如：

<p class=”a”></p>

不仅可以批处理所有class为a的标签，还可以在style里随意修改。

可以通过class属性给元素分组，相当于是把某些元素和某些元素分到了一个组里。

所以也有类选择器：

作用：根据元素的class属性值选中一组元素。

语法：.class属性值

例子：

.blue{

}

实际上，我们应该更常用class这样的类选择器。

并且，一个标签可以指定多个class值，这样就相当于一个属性被分类到了好几个类别中，比如这样：

<p class=”red abc”>111</p>

不过，在指定多个class属性的时候需要以空格隔开。

接下来是通配选择器

通配选择器会选择这个页面的全部元素，语法是*，写作这样：

*{

}
