有时候我们在使用类选择器的时候，可能需要综合多个条件来给样式，比如：

<div class=”red”>11</div>

<p class=”red”>11</div>

我们想要让div元素的字体变大，要怎么操作呢？

就可以用交集选择器，即：

Div.red{

}

交际选择器相当于放了类选择器和元素选择器的复合，必须同时满足既是div又是red才可以使用这个样式。

写法就是连着写，中间用.隔开：

语法：选择器1选择器2选择器3选择器4……

注意！！！

如果交集选择器里存在元素选择器，必须以元素选择器开头。

并集选择器的语法：   选择器1,选择器2

中间以逗号隔开

并集选择器会选择出现的全部选择器。在并集选择器里也可以插入id选择器。甚至可以写入交集选择器（作用是选择符合交集选择器的全部选择器）

例如：

#p,.p1,h1,span,div.red{

}
