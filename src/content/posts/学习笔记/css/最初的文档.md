# CSS 基础概述

## 网页的三个组成部分

- **结构**：HTML
- **表现**：CSS
- **行为**：JavaScript

## 什么是 CSS

CSS 的全称是 **层叠样式表**。

- 实际上我们看到的网页是多层的，能看到的只是最上层
- CSS 可以设置这些样式

## 行内样式

可以通过在标签内的 `style` 属性来设置样式：

```html
<p style="color:red;">内容文本</p>
```

### 基本语法

- 在样式的内部，用冒号来连接
- `style` 属性后面跟着的是样式，也就是 CSS 代码的内容

**语法格式：**
```
样式名: 样式值;
```

例如，设置段落文本颜色：
```html
<p style="color:red;">内容文本</p>
```

### 多个样式

CSS 样式可以像 HTML 属性一样连着写，在分号后面写空格然后连着写下一个样式：

```html
<p style="color:red; font-size:40px;">内容文本</p>
```

> **注意**：和 HTML 不同，CSS 必须要写单位，比如 `px` 不能省略。
