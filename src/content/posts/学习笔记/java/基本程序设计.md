# Java基本程序设计

## Java类的基本结构

每个Java程序都是从类的声明开始的，声明里的类名跟在关键字class后面：

```java
Public class ComputerArea{

    Public static void main(String[] args)
    {
        xxx
    }
}
```

例如：写一个计算圆的面积的程序：

```java
Double radius
Double area
```

## 变量创建

这样就创建好了变量，但是并不能输入，我们只学过`system.out.println`。

## 从控制台读入

### Scanner类

`System.in`是Java的默认输入，输入器是键盘，Java并不支持直接控制台输入，但是可以通过创建Scanner类的对象来实现读入输入，实现如下：

```java
Scanner input = new Scanner(System.in)
```

但这样没有给用户输入的提示，用户不知道应该输入什么，这种情况可以补加一条输出语句，提醒用户输入：

```java
System.out.println("enter radius")
```

### println与print的区别

`Println`和`print`很像，但两者拥有区别：
- `print`显示完后，不会把光标移到下一行
- `println`则会

Scanner读入后，再按下回车即读入完毕获得变量：

```java
Double radius = input.nextdouble();
```

### 导入包

如果想使用Scanner，需要导入util，如下：

```java
Import java.util.Scanner
```

也可以调用所有的类：

```java
Import java.util.*
```

## 赋值表达式

```java
System.out.println(x=1)
//等同于：
X = 1
System.out.println(X)
```

如果要把一个变量赋值给多个变量：

```java
I=k=j=l
```

这样的结果是：

```java
K = l
J = k
I = j
```
