从控制台读入：

System.in是java的默认输入，输入器是键盘，java并不支持直接控制台输入，但是可以通过创建Scanner类的对象来实现读入输入，实现如下：

Scanner input = new Scanner(System.in)

但这样没有给用户输入的提示，用户不知道应该输入什么，这种情况可以补加一条输出语句，提醒用户输入：

System.out.println(“enter radius”)

Println和print很像，但两者拥有区别。Print显示完后，不会把光标移到下一行，而println则会。

Scanner读入后，再按下回车即读入完毕获得变量：

Double radius = input.nextdouble();
