# 计算机，程序和Java概述

## 什么是计算机

计算机包含两个部分：
- 硬件
- 软件

### 计算机的主要硬件组件

- CPU
- 储存设备
- 输入设备
- 输出设备
- 通信设备

### 中央处理器（CPU）

中央处理器（CPU）从内存中获取指令并执行，CPU分为：
- **算数单元**：执行数值运算（加减乘除）和逻辑判断
- **控制单元**：控制协调其他组件

每台计算机都有一个内部时钟（这一点在操作系统会详细说明），时钟越快执行的指令就越多，时钟的计量单位是赫兹，1赫兹相当于每秒1个脉冲，现代计算机的时钟速度已经达到了千兆级。

最初开发的CPU只有一个核，核是处理器中实现指令读取和执行的部分。如今已经有两个或更多的核处理器，提高了CPU的处理能力。

## 比特和字节

计算机是一系列的电路开关，每个开关存在两个状态：
- 开（记作1）
- 关（记作0）

计算机内的所有数据都是以1和0进行储存的，这些1和0被称为**比特**（bit，二进制数）。

在计算机中，**字节**（byte）是最小的单元，由8个比特组成。

## 内存

内存是需要快速读取刷新的存储空间，不同于硬盘，内存的空间没那么大，但速度却很快，因此内存用来储存各种及时的信息数据供CPU使用。

特点：
- 当内存条断电后，就会损失所有信息（这一点不同于硬盘）
- 内存的地址的原始信息不一定为空，大多数时候是毫无意义的数据
- 一旦写入就可以正常使用

## 储存设备

内存是一个容易丢失的数据保存形式，一旦断电就会损失，所以程序和数据需要保存在储存设备上。目前储存设备主要有三种类型：

- **磁盘**：每台计算机至少有一个硬盘驱动器，可以永久的储存数据和程序
- **光盘**：分为只读和可读写光盘
- **USB闪存**：俗称U盘

## 输入输出设备

- **输入设备**：键盘、鼠标
- **输出设备**：显示器、打印机
