教学设计基本过程 期末复习重点 & 考点总结

一、核心考点梳理（按模块划分）

1. 教学设计核心模型（选择 / 简答高频）

ADDIE 模型：5 个阶段（分析→设计→开发→实施→评估），各阶段核心：分析（行为目标、受众等）、设计（课程 / 界面 / 序列）、开发（内容撰写 / 测试）、实施（教学落地 + 支持）、评估（形成性 + 总结性评价）。

ID4T 模型：9 大步骤（确定学习目的→编写具体目标→编排测试→分析学习起点→确定教材→创设活动→选择媒体→实施教学→评价改进），适用于中小学 / 幼儿园，强调动态调整。

2. 各阶段核心内容（简答 / 案例分析重点）

（1）分析阶段

三大核心：分析学习需要（现存问题、目标差距、总教学目标）、分析教学内容（加涅学习结果分类法归类）、分析教学对象（一般特征 + 个性特征：起点能力、学习风格、学习动机）。

关键考点：加涅五类学习结果（言语信息、智慧技能、认知策略、动作技能、态度），需区分各类型内涵。

（2）设计阶段

教学目标：表述三原则（系统性、层次性、具体性），包含四要素（学习者、行为、水平、条件）；三大领域（知识、技能、情感）及对应行为动词。

教学活动：情境设计（贴近生活、避免假情境）、分层教学（基础层模仿、中间层目标任务、优生前驱探索）。

（3）开发阶段

教学资源：含教学材料 + 工具，需结合教学目标和学生水平，拒绝照搬网络资源。

教学策略：开发三影响因素（内容类型：程序性知识用示范 - 模仿策略，启发性知识用启发式策略；教学环境；学生特征）。

（4）评价阶段

评价三要素：学习结果（学到什么）、学习过程（如何学）、学习特征（学习状况）。

评价类型：形成性评价（教学中，如课堂小测）、总结性评价（教学后，如课程考试）。

教学反思：三类内容（行为 / 观念 / 角色、设计 / 实施 / 结果、个性 / 风格 / 智慧）。

3. 高频题型预测

选择题：ADDIE/ID4T 模型步骤、加涅学习结果分类、教学目标表述原则、评价类型区分。

简答题：各阶段核心任务、模型内涵、反思分类、分层教学设计。

案例分析题：结合情境设计 / 分层教学 / 教学策略选择，分析合理性及改进措施。

二、必考名词解释（按考频排序）

教学设计：一种严谨的、用于设计课堂学习情境的方法，是教师运用系统方法确立学习目的、创设目标、分析学生、选择材料、开发活动、实施并改进教学的全过程。

ADDIE 模型：教学设计的传统模型，包含分析（analysis）、设计（design）、开发（develop）、实施（implement）和评估（evaluate）五个相互关联的阶段。

ID4T 模型：即 “教师教学设计模型”，由卡尔 — 查尔曼提出，含九大步骤，旨在帮助教师设计有效的课堂教学，强调动态调整。

形成性评价：在教学过程中（一段较短时间教学后）进行的评价，如课堂小测验，目的是及时了解教学进度、方法有效性，以便调整教学。

总结性评价：在较长时间教学后（如课程结束）进行的评价，如期末考试，用于反馈学习结果，为下次教学改进提供建议。

认知策略：学习者用以控制注意、学习、记忆、思考等行为的内在组织技能，需结合具体知识学习和应用。

动作技能：由操作规则和肌肉协调能力构成的习得技能，是一套操作规则支配下的流畅、合规则、准确的整体行为，可通过练习改进。

教学目标：学习者通过教学后获得的明确、具体的学习结果，是教学设计的出发点和归宿，需符合系统性、层次性、具体性原则。

分层教学：根据学生客观差异，将课堂练习分为三个梯度，让不同层次学生在 “最近发展区” 前进，满足不同学习需要的教学方式。

教学反思：教师对教学实践的结果进行审视、分析的过程，涵盖行为 / 观念 / 角色、设计 / 实施 / 结果、个性 / 风格 / 智慧三类内容。

评价改进教学：ID4T 模型的最后环节，是教师对教学过程和结果的总结与反思，目的是根据学习者、教学目标、社会需求等变化动态调整教学，提升教学效果。

三、简答题例题及标准答案（贴合期末考点）

例题 1

问：教学反思从内容上可分为哪三类？

答：① 关于行为、观念和角色层面的反思：对教学过程中的行为及背后指导理念进行分析；② 关于设计、实施以及达到的教学结果的反思：对教学方案设计、实施过程及效果进行分析；③ 关于个性、风格及智慧的反思：对教学中问题处理所体现的个性、智慧进行分析。

例题 2

问：教学目标的表述有哪些原则？

答：① 系统性原则：教学目标是学科、单元、课时目标构成的上下贯通的完整体系，范围需适宜；② 层次性原则：结合教学内容难易和学生能力水平，做到目标与学习者层次匹配；③ 具体性原则：明确学习者、行为、掌握水平、完成条件四要素，可测量、可观察。

例题 3

问：分层教学的实施步骤要点是什么？

答：① 学情分析：明确学生在起点能力、学习风格、学习动机上的差异；② 梯度设计：课堂练习分三个层次，基础差学生以模仿性学习为主，主动性差学生设目标性任务，优生采用任务驱动自主探索；③ 教学调控：对优生 “放中有扶”（侧重自学指导），对中等生和后进生 “扶中有放”（侧重带领学习），确保各层次学生获得成功体验。

例题 4

问：简述 ADDIE 模型的五个阶段及核心任务。

答：① 分析阶段：分析行为目标、教学任务、受众、环境等，明确教学问题和目标差距；② 设计阶段：对课程、界面、教学序列等进行整体设计；③ 开发阶段：撰写课程内容、设计页面、进行教学测试，完善教学材料；④ 实施阶段：开展教学活动，提供实施支持；⑤ 评估阶段：通过形成性评价和总结性评价，判断教学效果。

例题 5

问：教学策略的开发受哪些因素影响？请简要说明。

答：受三个核心因素影响：① 内容类型：程序性知识宜用示范 — 模仿、先行组织者策略；启发性知识宜用启发式、协作学习策略；② 教学环境：支持合作 / 实验的环境可采用协作、发现学习策略，传统教室需选择适配的策略；③ 学生特征：需结合学生的经验水平、角色喜好、知识掌握能力等设计策略，避免多数学生不适应。

例题 6

问：加涅的五类学习结果分别是什么？请简要解释。

答：① 言语信息：学习者学会陈述事实或观点的能力，是其他学习的先决条件；② 智慧技能：应用符号办事的能力，从低到高分为辨别、概念、规则、高级规则；③ 认知策略：控制自身学习、记忆、思考等行为的内在组织技能；④ 动作技能：由操作规则和肌肉协调能力构成的整体行为，可通过练习改进；⑤ 态度：影响人对事物作出个人选择的内部状态，主要通过模仿和成功体验习得。
