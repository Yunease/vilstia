# 教育心理学

## 名词解释

### 基础概念

- **心理学**：心理学是一门研究心理和行为的科学
- **注意**：注意就是心理活动对一定对象的指向和集中
- **无意注意**：也叫不随意注意，是指事先没有预定目的、也不需要意志努力的注意
- **有意注意**：也叫随意注意，是服从于预定目的、需要作意志努力的注意
- **有意后注意**：也叫随意后注意，是注意的一种特殊形式，是指事先有预定的目的，不需要意志努力的注意

### 感知与记忆

- **感觉**：感觉是人脑对直接作用于感觉器官的客观事物的个别属性的反映
- **知觉**：知觉是直接作用于感觉器官的事物的整体在脑中的反映，是人对感觉信息的组织和解释的过程
- **记忆**：记忆是人脑对过去经历过的事物的反映。用信息加工的语言来讲，记忆是对信息进行编码、储存和提取的过程
- **识记**：识记是识别和记住事物，从而获得和积累知识经验的过程
- **保持**：保持是识记过的事物以一定的形式储存在人的头脑中，并得到巩固的过程
- **再现**：再现是从记忆加工中提取信息的过程
- **回忆**：过去经历过的事物形象或概念在头脑中重新浮现的过程叫回忆，也称重现

### 思维与想象

- **思维**：思维是借助于言语、表象和动作实现的，对客观事物的本质和规律的概括的间接反映
- **创造性思维**：创造性思维是重新组织已有的知识经验，提出新的方案和程序，并创造出新的思维成果的思维活动
- **概念**：概念是人脑反映客观事物的本质特征的思维形式
- **思维定势**：思维定势又称为心向，它是由先前的心理操作形成的模式所引起的心理活动的准备状态，也就是人们在过去经验的影响下，解决问题时的倾向性
- **功能固着**：人们把某种功能赋予某种物体的倾向称为功能固着

### 想象

- **想象**：人不仅能对直接感知的事物形成知觉形象和记忆表象，而且还能反映未曾感知过的事物，并在头脑中形成关于它们的形象，这就是想象
- **无意想象**：没有预定目的和任务，不由自主产生的想象
- **有意想象**：按一定目的自觉进行的想象
- **再造想象**：根据言语的描述或图样的示意在人脑中形成相应新形象的过程
- **创造想象**：在创造活动中根据一定的目的任务，在人脑中独立的创造出新形象的心理过程
- **幻想**：指向于未来并与个人的愿望相联系的想象

### 情绪与人格

- **动机**：激发和维持个体进行活动，并导致该活动朝向某一目标的心理动力
- **情绪**：情绪是人受到情景刺激，经过是否符合自己需要的判断后，而产生的行为变化、心理变化和对事物或事件态度的主观体验
- **情绪状态**：情绪状态是在产生情绪时不同程度的体验
- **心境**：心境是一种比较微弱而持久存在的具有渲染性的情绪状态
- **激情**：激情是一种剧烈的、短暂的、爆发式的情绪状态
- **应激**：应激是指在出乎意料的情况下所引起的情绪状态
- **人格**：人格是探讨完整个体与个体差异的领域
- **气质**：气质是人格结构中比较稳定、与遗传素质密切相关的成分，也就是我们平时所说的脾气、秉性
- **性格**：性格是一个人在现实生活中形成的对现实比较稳固的态度以及与之相适应的习惯化了的行为方式

### 自我与学习

- **控制感**：控制感是人类的一个基本内在动机，这种动机表现在人类希望能够通过自己的行为预知和改变环境，希望行为能够产生自己渴望得到的结果
- **自我效能感**：自我效能就是对能力的知觉，相信自己拥有个人技能和工作能力，使我们在特定的情境中正确而成功的行动
- **学习**：学习是个体在经验的基础上引起的相对持久的心理结构或外显行为的变化
- **观察学习**：观察学习是指通过他人行为及其强化性结果的观察而获得某些新反应的学习过程
- **学习迁移**：已获得的知识、技能、方法或态度对学习新内容和解决新问题所产生的影响就是学习迁移
- **认知风格**：认知风格也叫认知方式或认知类型，是指个体在加工信息时（包括接收、贮存、转化、提取和使用信息）的心理倾向
- **学习风格**：学习风格也称学习方式或学习偏好，是人们在学习时对不同学习条件的偏好和需求，它是个体在探究解决其学习任务时表现出典型的、一贯的、独具有个人特色的方式

### 健康与辅导

- **健康**：健康不仅是没有疾病，而且包括躯体健康、心理健康、社会适应良好和道德健康
- **心理辅导**：学校教育者根据学生心理发展的特性与规律，运用心理学等专业知识技能，通过集体辅导、个别辅导等多种形式设计，帮助学生消除成长中面临的障碍，形成良好的心理素质，充分发挥个人潜能，促进其心理健康与人格和谐发展的过程

## 简答与论述

### 心理学的研究任务

1. **描述心理与行为事实**（是心理科学研究的第一步）
2. **揭示心理与行为发生的机制**（是心理科学研究的主要任务）
3. **干预心理与行为**（是心理学研究最高层次的任务）

### 心理学研究的一般过程

1. 发现问题，选择课题
2. 提出假设，设计研究
3. 实施研究，收集数据
4. 分析数据，检验假设
5. 撰写报告，评价研究

### 师范生学习心理学的意义

#### 一、掌握必要的心理学知识是师范学生未来从事教育教学工作的必要条件

掌握心理学的原理和方法，有助于解决如何教和怎样教得更好的问题，也有助于将来从心理学的角度分析、研究和解决教书育人过程中出现的其他各种问题。

教师担负着教书与育人的双重任务。要培养学生健康人格和良好的品德。教师必须了解学生人格和品德形成与发展的规律。当代心理学研究所提出的关于个体人格和品德形成与发展的基本原理、规律是学校实施思想品德教育的重要理论基础，同时也能够为检验学校德育的实效性和科学性以及学生品德的评定提供具体的科学指导和依据。

#### 二、学习心理学有助于大学生更好的适应大学生活及自身健康发展

如何积极适应大学阶段的学习生活，有效的应对专业学习、日常生活以及个人发展中遇到的困扰与问题是当前大学生面临的一个重要问题。通过心理学的学习，掌握人的心理活动特点和规律，有助于大学生深入的认识自己、了解他人，学会自我调节的方法，提高自身的心理素质，更好的应对自己大学生活中的挑战和任务，从而实现个人在专业学习和其他方面的更好发展。

学习心理学，有助于大学生更深入的掌握学习的规律与特点，获取有效的学习以及创造性解决问题的策略和方法，提高学习能力与效率，培养和发展自己的创造力。

### 引起无意注意的原因

#### 一、刺激物的特点

1. 刺激物的强度
2. 刺激物之间的对比关系
3. 刺激物的活动与变化
4. 刺激物的新异性

#### 二、主体自身的状态

1. 需要和兴趣
2. 情绪和过去经验
3. 人的身体状态

### 有意注意的维持条件

1. 加深对活动任务的理解
2. 努力排除与活动任务无关的干扰
3. 培养间接兴趣
4. 增强克服困难的意志力

### 注意规律教学

#### 一、精心设计教学导入，先声夺人

吸引学生的无意注意，激发学生的学习兴趣，使课堂教学取得事半功倍的效果。

一般有如下做法：
- 创设情境
- 设置疑问
- 以旧引新

#### 二、运用丰富灵活的教学方法，新颖有趣

激发学生的无意注意，使学生产生浓厚的求知欲望。具体做法是：
- 优化板书设计
- 运用有趣的演示实验
- 提高教学的语言艺术

#### 三、创造性的挖掘教材内涵，深化教学

应根据教学内容来确定，要避免千篇一律，把握尺度是关键

#### 四、巧妙的构思结束语，耐人寻味

收拢学生纷繁的思绪，激发学生的求知欲，做到课虽终而意无穷

#### 五、避免无关刺激的干扰，优化课堂

教师要注意：
- 仪表大方得体
- 板书设计应布局合理，层次清楚，富有章法
- 教具出示要及时

#### 一、利用感知规律，增加学生的感性认识

#### 三、运用感知规律，培养学生的感知能力

2. 注意观察与思考相结合
3. 注意多种感官相结合

### 影响识记效果的因素

3. 识记态度

### 记忆规律在学习中的应用

#### 一、运用记忆规律，有效组织复习

1. 明确实际目的，增强学习主动性
2. 理解学习材料的意义，建立知识间的内在联系
3. 运用各种记忆策略，促进对知识的保持
4. 注意复习方法，防止知识遗忘

### 如何有效的掌握概念

1. 促进学生对特定领域知识的掌握
2. 要培养学生主动提出问题和解决问题的内在动机
3. 问题的难度要适当
4. 加强元认知能力的培养

### 影响学习迁移的因素

#### （一）影响学习迁移的主观因素

1. **智力水平**：智力对迁移的质和量都有重要作用
2. **原有经验的组织性及可利用性**：知识经验的组织性一方面体现在其抽象、概括性方面，另一方面表现在经验的丰富性方面
3. **学习的定势**：由于定势是个体在认知方面和外显的行为方面以一种特定的方式进行反应，使个体在思维和活动方向的选择方面有一定的倾向性，因此，定势在迁移的过程中有一定的作用
4. **元认知水平**：元认知就是个体以自己认知过程的活动为对象的认识。元认知对整个学习过程起着监控和调节的作用

#### （二）影响学习迁移的客观因素

1. **学习任务情境的相似性**：两种学习任务情境之间的相似性，包括在学习内容、学习条件、学习目标与学习过程等方面是否具有影响迁移发生的共同要素
2. **教学方法**：教师在教学时是否有意识的引导学生发现不同知识之间的共同点，启发学生将习得的原理和技能应用于新的情境，指导学生监控自己的学习对迁移的发生有重要影响
3. **练习**：如果学生在学习基本概念和原理的同时给予充分的练习时间及多样化的练习情境，可以使他们对知识的掌握牢固而清晰，在熟能生巧的基础上产生迁移

### 学习迁移的有效促进

#### （一）合理编排教学内容

教材内容的选择及编排直接影响学生认知结构形成的特点。为使学生形成知识间的丰富联系，建立良好的认知结构。教材内容首先应该体现这种知识结构，使教材内容结构化、一体化、网络化

#### （二）合理处理教学程序

教学程序是使用有效的教材发挥功效的直接环节。合理安排教学程序应从宏观、微观两个方面加以考虑

#### （三）教授学习方法，引导学生学会学习

学习方法是促进有效学习的一些手段、措施和技巧，它本身就包含了有效迁移的信息，可以在后续的学习中产生比较广泛的一般性迁移

#### （四）提高学生的元认知水平

教师在实际教学中要有意识的向学生传授一些元认知策略，使学生掌握策略的使用条件，并要求学生对策略使用的成败进行自我评价，将有助于学生学会如何学习

#### （五）培养思维的灵活性，克服定势的消极影响

在安排学习课题时，要把同一性和变异性相结合，既要考虑所授内容与学生原有经验的一致性，同时又要变化课题，培养学生思维的灵活性，克服定势的消极作用

#### （六）创设丰富情境，拓展知识的应用范围

在教学中要创造条件尽量使学生在真实的任务情境中进行学习，在各种不同的情境中练习所学之所学习的知识、技能，从不同角度去理解、应用知识，这是促进学习广泛迁移的主要途径

### 心理健康的标准

#### （一）一般标准

1. **认识正常**：这里的认识正常包含两层意思，首先是智力发展正常。其次，认识正常是指个体基本的认识活动过程正常，能够正确的反映外部世界，正视现实、接受现实，与客观现实能够保持较好的同一性
2. **情绪稳定乐观**：情绪稳定乐观是心理健康的主要指标
3. **意志品质健全**：意志品质健全是指个体在行动过程中拥有自觉性、果断性、自制性和坚持性等良好意志品质
4. **行为表现正常**：心理健康的人行为表现协调，反应适度
5. **人际关系和谐**：良好的人际关系是维护心理健康不可缺少的条件
6. **人格独立完整**：人格也称个性，是个体区别于他人的稳定、独特的心理特征的总和
7. **自我意识正确**：自我是人格的核心与统帅
