# 1.1 随机事件

- 可重复性（可以重复进行）
- 可预知性（知道有几个结果）
- 不确定性（不知道是哪个结果）

复合这些条件的，被称为随机实验，标记为E，如投色子，投篮。



## 2. 样本空间

随机实验的所有可能结果构成的集合被称为样本空间，称为S，例如在投硬币中：

S = { 正，反 }



## 3. 样本点

样本空间的每一个元素被称为样本点，记为e。



## 4. 随机事件

样本空间的子集被称为随机事件，即为A，B，C。比如：

S = { 1,2,3,4,5,6 } A = { 1,3,5 } = "投掷筛子出现奇数点"

- 事件A的发生：只需要某个样本点出现，就认为是事件A发生。例如投出了3。
- 必然事件：必然发生的事件。
- 不可能事件：也就是空集。



## 例题：写出下列试验的样本空间

(1) \(E_1\)：抛一枚硬币两次，观察正反面朝上的情况。（正面：H，反面：T）

- { HH，HT，TH，TT }

(2) \(E_2\)：抛一枚硬币两次，观察出现正面的次数。

- { 0，1，2 }

(3) \(E_3\)：某市急救中心的电话台一夜中接到的呼唤次数。

- { 0，1，2... }



# 1.2 事件之间的关系和运算

## 1. 事件的包含

定义：当事件A发生，则必导致事件B发生。



## 2. 事件的和

定义：A发生或B发生（A∪B），或是A，B至少发生了一个。

A+B+C ：A，B，C这三个事件中，至少发生了一个。



## 3. 事件的积

定义：AB或A∩B

当事件A发生而且B发生。

当事件A，B同时都发生。



## 4. 事件的差

定义：A-B

事件A发生，但B不发生。如果A-B-C，那么意味着事件A发生，B，C都不发生。

**重要公式**：A-B = A-AB



## 5. 互不相容事件-互斥事件

定义：A与B互斥，也就是A*B = 空集



## 6. 对立事件（互逆事件）

A与B对立->AB = 0，并且A∪B = S全集。其中一方必须发生，一个发生另一个一定不发生。



## 7. 事件运算满足的定律：

- 交换律：A∩B = B∩A，A∪B = B∪A
- 结合律：（A∪B）∪C = A∪（B∪C），（A∩B）∩C = A∩（B∩C）
- 分配律：（A∪B）∩C = （A∩C）∪（B∩C），（A∩C）∪B = （A∪C）∩（B∪C）

- 德摩根率：

$$
\overline{A \cup B} = \overline{A} \cap \overline{B},\quad \overline{A \cap B} = \overline{A} \cup \overline{B}
$$



## 例题：设 \(A,B,C\) 是三个事件，试用 \(A,B,C\) 表示下列事件：

(1) \(A,B\) 都发生，C 不发生。

- A∪B∪反C

(2) \(A,B,C\) 恰有一个发生。

- 反A反BC ∪ A反B反C ∪ 反AB反C

(3) \(A,B,C\) 至少发生一个。

- A∪B∪C

(4) \(A,B,C\) 至少发生两个。

- A反BC ∪ 反ABC ∪ AB反C ∪ ABC



# 概率的定义与性质

## 1. 概率的统计性定义

### （1）频率

设E是一个随机实验，A为任意事件，把E独立重复n次，Na表示事件A发生的次数，那么称 Na/N = fn(A) 为A发生的频率。

### （2）概率

当实验次数N充分大的时候，频率fn(A)→P，那么称常数P为A发生的概率，即为P（A） = P
