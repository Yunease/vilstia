# 计算机网络 期末试卷4的复习

## 选择题

1. **( A )** 被认为是世界上第一个计算机网络
   - **答案**：A．ARPAnet

2. 设某段电路的传播时延是10ms，带宽为10Mb/s,则该段电路的时延带宽积为 **( C )**
   - **答案**：C．1×10⁵bit
   - **解析**：时延带宽积 = 传播时延 × 带宽 = 10ms × 10Mb/s = 10⁻² × 10⁷ = 10⁵ bit

3. 在OSI模型中，提供流量控制功能的层是第 **( B )** 层
   - **答案**：B．2、3、4

4. 在OSI模型中，提供建立、维护和拆除端到端连接的层是 **( D )**
   - **答案**：D．传输层

5. 以太网交换机是按照 **( A )** 进行转发的
   - **答案**：A．MAC地址

6. 以下的以太网电缆标准中，**( B )** 是使用光纤的
   - **答案**：B．10BaseF

7. 电路交换的优点有 **( A )**
   - **答案**：(1)和(2)
   - **解释**：传输时延小、分组按序到达

8. 下列协议中，不会发生碰撞的是 **( A )**
   - **答案**：A．TDM
   - **解释**：TDM是时分复用，严格按照时隙传输，不会碰撞

9. PPP协议提供的功能有 **( D )**
   - **答案**：D．A、B和C都是
   - **解释**：包括成帧方法、LCP链路控制协议、NCP网络控制协议

10. 以下哪个设备是工作在物理层 **( C )**
    - **答案**：C．集线器
    - **解释**：路由器(网络层)、交换机(数据链路层)、网桥(数据链路层)

## 知识点速记

### 以太网标准

| 标准 | 传输介质 | 最大段长 |
|------|---------|---------|
| 10Base5 | 粗同轴电缆 | 500m |
| 10Base2 | 细同轴电缆 | 185m |
| 10BaseT | 双绞线 | 100m |
| 10BaseF | 光纤 | 2000m |

### 流量控制层

- 数据链路层
- 网络层
- 传输层

### 电路交换特点

- **优点**：传输时延小、分组按序到达
- **缺点**：需要建立连接、线路利用率低

### 介质访问控制方法

| 方法 | 是否碰撞 | 说明 |
|------|---------|------|
| TDM | 不会 | 时分复用，严格时隙 |
| ALOHA | 会 | 随机接入 |
| CSMA | 会 | 载波监听多路访问 |
| CSMA/CD | 会 | 冲突检测 |

### PPP协议功能

- **成帧方法**：定义数据帧的格式
- **LCP**：链路控制协议，用于建立、配置和测试数据链路连接
- **NCP**：网络控制协议，用于建立和配置不同网络层协议

### 网络设备层次

| 设备 | 工作层次 |
|------|---------|
| 中继器 | 物理层 |
| 集线器 | 物理层 |
| 网桥/交换机 | 数据链路层 |
| 路由器 | 网络层 |
