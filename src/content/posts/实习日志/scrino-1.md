# 前言

正式实习的第一天，我基本上要带三年级的小学生，主要学习scrino这个软件的应用，因此开一个专栏，讲讲scrino。



# 功能分区

scrino分为模块区，脚本区，演示区和舞台区。

模块区会根据积木的功能，区分不同的颜色，例如蓝色是运动积木等。在实际的教学工作中，应该给学生讲解不同颜色积木的基本用途。

脚本区是积木拼接的地方，积木具有凹槽和凸起，可连接的积木靠近时，会出现灰色阴影。拼接完毕后，点击任意积木，就可以运行整体脚本。再次点击可以停止。

演示区就是最后执行出来的动画啦，没什么好说的。

舞台区是素材库，分为角色和背景。角色会拥有一些基本属性，如：

- x
- y
- size
- rotation

这些基本属性都可以通过积木来调整控制。每一个角色，舞台分别应用不同的脚本。

# 初次编程

拖拽模块区的积木到脚本区，就算正式加载脚本了，此时点击任意积木，就会执行与之相连的所有积木内容。再次点击会停止执行。

控制模块中的重复执行积木类似while循环，可以一直执行。

# 控制程序运行

scrion软件拓展到全屏模式时，可以看到一个小绿旗和小红圆。

想要点击绿色旗帜运行脚本，需要先添加控制积木，在`控制`中找到`当🚩被点击`，拖拽进脚本区并于脚本连接后，现在点击小旗帜，程序就能运行了——怎么感觉都是废话。

# 边缘反弹

在动画的运行中，我们希望图像到边缘的时候可以反弹回去，防止出现图像超出屏幕边缘的情况。

在`运动`模块中，有一块积木是`碰到边缘并反弹`，注意在添加这个积木的同时，应该添加`将翻转方式设置为 左右反转`，否则会发生全翻转，积木会倒立。

最终搭建的积木应该如下：

```python
当🚩被点击：
	将翻转方式设置为 左右翻转
	重复执行：
		移动10步
        碰到边缘就反弹
```
