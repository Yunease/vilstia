# 前言

正式实习的第一天，我基本上要带三年级的小学生，主要学习scrino这个软件的应用，因此开一个专栏，讲讲scrino。



# 功能分区

scrino分为模块区，脚本区，演示区和舞台区。

模块区会根据积木的功能，区分不同的颜色，例如蓝色是运动积木等。在实际的教学工作中，应该给学生讲解不同颜色积木的基本用途。

脚本区是积木拼接的地方，积木具有凹槽和凸起，可连接的积木靠近时，会出现灰色阴影。拼接完毕后，点击任意积木，就可以运行整体脚本。再次点击可以停止。

演示区就是最后执行出来的动画啦，没什么好说的。

舞台区是素材库，分为角色和背景。角色会拥有一些基本属性，如：

- x
- y
- size
- rotation

这些基本属性都可以通过积木来调整控制。每一个角色，舞台分别应用不同的脚本。
